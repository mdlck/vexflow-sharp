## Font Software And Generated Font Data

`VexFlowSharp.Common` includes generated C# font metric and outline data derived
from VexFlow font packages. Unity package folders also include font binaries for
runtime UI Toolkit text rendering.

The generated C# files are not a replacement for the original font licenses.
When distributing VexFlowSharp, keep this notice with the generated font data
and bundled font binaries.

are included in a distributed Unity package, include the relevant upstream
